# N7 - Digital Banking Frontend Implementation 🚀

A pixel-perfect, fully responsive frontend implementation of the N7 Digital Banking platform, developed as an assignment. The focus of this project is on high-fidelity design translation from Figma, modern frontend architecture, and smooth, engaging user interactions.

## 🔗 Live Demo
[**Click here to view the live project**](https://n7-frontend-assignment-two.vercel.app/) 

## 🛠️ Tech Stack
* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS (for rapid, utility-first pixel-perfect styling)
* **Animations:** Framer Motion (for 3D carousels and smooth scroll reveals)
* **Icons & Assets:** Custom SVGs and optimized PNGs

## ✨ Key Features & Technical Highlights

* **Pixel-Perfect Figma Translation:** Meticulous attention to spacing, typography, padding, and layout alignments to exactly match the provided Figma designs.
* **3D Interactive Case Study Slider:** Built a custom 3D coverflow carousel using `framer-motion`, featuring smooth layout transitions, realistic scale/rotation, and dynamic rendering of company logos and content.
* **Complex CSS Animations:** * Implemented a continuous, seamless "Marquee/Ticker" for partner logos.
  * Engineered a custom **sweeping gradient animation** (circular 4-corner transit) on the massive N7 footer watermark using arbitrary Tailwind values and CSS keyframes.
* **Advanced Hover States:** Developed intricate hover interactions, including animated directional underlines and smooth color-flipping CTA buttons (gradient-to-solid transitions).
* **Fully Responsive:** Fluid layouts that adapt seamlessly across Desktop, Tablet, and Mobile devices using CSS Grid and Flexbox.
* **Clean & Maintainable Code:** Component-driven architecture ensuring reusability and scalability.
